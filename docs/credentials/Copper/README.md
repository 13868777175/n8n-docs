# Authentication

## Pre-requisites

Create a [Copper](https://www.copper.com/) account.

## Steps

### API KEY

1. Open your Copper dashboard.
2. Click on Settings.
3. Click on Integrations, API Keys.
4. Generate API key.
5. Use selected API key in your Copper node credentials in n8n.
