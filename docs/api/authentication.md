## Create an API key

1. Log in to n8n.
2. Go to **Settings** > **API**.
3. Select **Create an API key**.

## Delete an API key

1. Log in to n8n.
2. Go to **Settings** > **API**.
3. Select **Delete** next to the key you want to delete.
